LCS: Longest Common Subsequence

The LCS for two given sequences is defined as the largest sequence such that it is a subsequence of the two sequences.


What's a Sequence?

A sequnce is a set of ordered characters.


What's a Subsequence?

A subsequence is a portion of an original sequence that satisfies the given property:
For a sequence X = {x1, x2, ..., xn}, any subsequence Y derived from X, for any two elements xi, xj, if xi appears before xj in X, then xi must also appear before xj in Y.
